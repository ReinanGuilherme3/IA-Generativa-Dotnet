🤖 IA Generativa com .NET


Conceitos, exemplos práticos, arquiteturas e boas práticas para construção de aplicações inteligentes com .NET.



O foco não é apenas aprender bibliotecas específicas, mas compreender os conceitos por trás da IA Generativa e aplicá-los de forma consistente em aplicações reais desenvolvidas com .NET.


🎯 Objetivo

Este repositório reúne exemplos progressivos e práticos sobre IA Generativa, organizados do conceito mais fundamental até arquiteturas mais complexas. Cada exemplo é independente e acompanha seu próprio README com explicações detalhadas.
